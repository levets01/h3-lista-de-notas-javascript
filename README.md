# 📝 Notas App

Una aplicación web sencilla para crear, visualizar y eliminar notas, con almacenamiento en el navegador usando **localStorage**.

---

## 🚀 Características

- Agregar notas rápidamente
- Eliminar notas individuales
- Persistencia de datos con `localStorage`
- Diseño moderno y responsive
- Animaciones suaves al agregar notas
- Interfaz minimalista y agradable

---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3 (variables, flexbox, animaciones)
- JavaScript (vanilla JS)
- Web Storage API (`localStorage`)
